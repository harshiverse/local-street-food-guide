# Local Street Food Guide

A simple web app that recommends street food based on local culture, weather, and cravings.

## Tech Used
- HTML
- CSS
- JavaScript

## How It Works
Users select their mood and weather.
The app suggests street food based on local eating habits.

## Why This Exists
This project demonstrates how local context can improve recommendations.
It uses a custom `.kiro/product.md` file to teach cultural behavior.
